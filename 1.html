HTML이란 무엇인가?

Hypertext Markup Language(HTML) is the standard markup Language for creating web pages and web applications.
It is a type of web language used to create documents that can be viewed through the World Wide Web, one of the Internet services. In particular , it was developed to write hypertext , and most web pages accessed through the web on the Internet are written in HTML .

HTML is a command that defines the text size, color, shape, graphics, and movement (hyperlink) of a document, and is used to create a homepage. Commands used in HTML are called tags , and are indicated using angle brackets "<>". In general, tags consist of two pairs that indicate the start and end, but in some cases, such as tags such as " img " and " br ", the effect is expressed only with the start tag, and the name of the end tag is a slash character [/]. It begins. A document written in HTML like this is called an HTML document, and a document written in HTML is interpreted by a web browser and displayed to the user. In HTML , a document recognizes a separate code , creating complete hypertext, as well as linking words or short sentences to other places or files on the Internet.

HTML was created to define the format of electronic documents and was defined as a subset of the international standard SGML . HTML is a language created by emphasizing hypertext in SGML , and is composed of plain text in ASCII code . This language does not require a separate compiler and is in the limelight as an easy-to-use language that can be interpreted by web browsers.
